# Rediseño sitio Residencial MiraVerde 🌿

Este proyecto es un ejercicio personal de rediseño del sitio web de la residencial MiraVerde. El objetivo es mantener su identidad visual, pero modernizar su presentación con técnicas actuales de maquetación y diseño.

## 🧱 Estructura del proyecto
- `blocks/`: Estilos CSS separados por componentes (navbar, hero, etc.)
- `images/`: Imágenes del proyecto (logo, hero, etc.)
- `pages/`: Páginas adicionales si las hay
- `index.html`: Página principal

## 🚧 En desarrollo
Actualmente en etapa inicial: se han implementado el `header` con navbar y una sección tipo "hero".

## 💡 Tecnologías usadas
- HTML5
- CSS3 con `@import` modular

"Este proyecto es un ejercicio personal de rediseño sin fines comerciales. No está afiliado a la residencial original."
